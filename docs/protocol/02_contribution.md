# Contribution Statement

This document states precisely what new knowledge this project adds to the field. It is written before analysis begins and is the standard against which the project's success is measured — not whether the hypotheses are confirmed, but whether the questions are answered rigorously.

A null result that is adequately powered and correctly interpreted is a contribution. A positive result from an underpowered or biased study is not.
